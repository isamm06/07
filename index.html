Trabajo 05 - Uso de tablas

1.¿Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

Table es la etiqueta principal porque es la encargada de contener todo lo demas, es decir, es la que marca el inicio y el final de la tabla

2.¿Qué etiqueta se utiliza para definir una fila en una tabla?

Tr es la etiqueta que se utiliza para definir una fila en una tabla

3.¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

Th es la etiqueta que se utiliza para crear una celda de encabezado

4.¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

Td es la etiqueta que se utiliza para crear una celda de datos

5.¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

El atributo colspan se utiliza para especificar cuantas columnas debe abarcar

6.¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

El atributo rowspan es el que se utiliza para especificar cuantas filas debe abarcar
